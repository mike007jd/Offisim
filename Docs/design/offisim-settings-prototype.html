Settings · functional specimens

The window above is the canonical resting Settings (Pi Agent tab, runtime status visible, no dirty state). Every specimen below is built 1:1 with the real React surface and its controlling hook — no invented controls, no dropped states. Settings is not a wizard, so every pane stays on the light surface; only Wizards may use --wiz-*. Specimens cover gaps the resting window cannot show: External Employees (empty + populated + install wizard), MCP (grouped list + add form + stdio confirm), Runtime (density/theme/switches/binding/ diagnostics), Pi Agent (auth status, model registry, session storage), save bar transitions, discard toast, validation reject, and desktop vault snapshot export.
