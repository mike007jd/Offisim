Global lifecycle — state specimens

Phase 6 is not a single workspace — it is the cross-cutting lifecycle layer that wraps every workspace: first-run welcome → company create/select → onboarding tour → project lifecycle → deliverable management → employee onboarding. The window above is the canonical first thing a new user sees: FirstRunWelcomeScreen (DialogShell size="xl" closeOnBackdrop=false) over a bare, dimmed shell. Each lifecycle surface — built 1:1 with the real component + its controller — is below.
