# Offisim 1.1.1

Offisim 1.1.1 is a security and reliability update for the engine-neutral macOS desktop workspace.

## Highlights

- Fixed install deep links so signed desktop builds reliably open the requested package flow.
- Hardened Codex executable discovery and launch against path substitution and time-of-check/time-of-use attacks.
- Hardened vault, secret-key, package, updater, ZIP, workspace, checkpoint, shell-lifetime, and skill-overlay filesystem boundaries.
- Removed the remaining high-severity JavaScript parser, command-execution, and actionable Rust path findings reported by GitHub CodeQL.

## Distribution

- Apple silicon macOS build.
- Developer ID signed and Apple notarized.
- SHA-256 checksums are included with the release assets.
