# Offisim 1.1.2

Offisim 1.1.2 is a patch release for installed-app Codex reliability and applicable dependency security advisories.

## Highlights

- Restored signed installed-app Codex discovery and real task execution.
- Launches a private lifetime-bound copy of the validated native executable so the existing no-symlink and TOCTOU defenses remain intact.
- Updated applicable Hono, Rust networking/serialization, and asset-processing/tooling dependencies for the corresponding Dependabot advisories.

## Distribution

- Apple silicon macOS distribution.
- Production release requires Developer ID signing and Apple notarization.
- SHA-256 sidecars are required for release assets.
